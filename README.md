# misc-labs

This repository contains miscellaneous directories that I use to spin up labs, run demos, etc.

Each sub-directory contains, or will contain, the README.md explaining the intent of its content. Either Terraform labs, demo scripts, or other.

For Terraform-based labs, look for a `terraform.tfvars.sample` file (for example under `landing-zone/`) to understand the variables you need to provide before running `terraform apply`.
